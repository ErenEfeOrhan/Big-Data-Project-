# Dataset Kurulumu

Bu proje Spotify Tracks Dataset kullanır.

## Dataset Nasıl Eklenir?

1. Kaggle'dan Spotify Tracks Dataset indir
2. CSV dosyasını şu isimle kaydet:

spotify_tracks.csv

3. Bu klasöre koy:

data/spotify_tracks.csv
