# Project Screenshots and Graph Outputs

Bu klasör proje teslimi ve sunum için kullanılan ekran görüntülerini ve grafik çıktıları içerir.

## Klasörler

- `manual/`  
  Elle alınan ekran görüntüleri:
  - Docker servisleri
  - Producer logları
  - MLflow ekranı
  - Dashboard ekranı
  - GitHub commit geçmişi

- `eda/`  
  EDA aşamasında üretilen grafikler.

- `ml/`  
  Makine öğrenmesi aşamasında üretilen model karşılaştırma, feature importance,
  actual vs predicted ve residual grafik çıktıları.

## Açıklama

Bu klasördeki görseller, proje çıktılarını ve sunum kanıtlarını desteklemek amacıyla saklanmaktadır.
